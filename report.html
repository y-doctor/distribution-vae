Distribution VAE

Compressing single-cell expression distributions from Perturb-seq into fixed-size latent vectors via a Variational Autoencoder

1. The Problem: Single-Cell RNA-Seq & Perturb-seq

In single-cell RNA sequencing (scRNA-seq), we measure the expression level of thousands of genes in individual cells. Unlike bulk RNA-seq which averages over millions of cells, scRNA-seq reveals cell-to-cell variability — the full distribution of expression, not just a single number.

Perturb-seq combines CRISPR perturbations with scRNA-seq: cells receive different genetic perturbations (e.g., gene knockouts), and we read out the resulting expression profiles. For each (gene, perturbation) pair, we get a distribution of expression values across all cells that received that perturbation.

Figure 1: Perturb-seq produces a distribution of expression values for each (gene, perturbation) pair.

The core challenge: Each (gene, perturbation) pair produces a variable-length set of expression values (one per cell in that group). We need a fixed-size representation to feed into downstream ML models. Simply taking the mean discards all distributional information — shape, skewness, multimodality.

2. The Key Insight: Quantile Functions

The project's central idea is elegant: sorting is all you need.

A sorted empirical sample is the quantile function (inverse CDF) of the distribution. By interpolating the sorted values onto a fixed-size grid of K=256 points, we get a canonical, fixed-dimensional representation that:

  • Is permutation-invariant — order of input cells doesn't matter
  • Is order-preserving — monotonically non-decreasing by construction
  • Works with any number of input cells (20, 200, 2000...)
  • Captures the full distributional shape, not just summary statistics

Figure 2: Converting variable-length expression values into a fixed-size quantile grid by sorting and interpolating.

3. VAE Architecture

The fixed-size quantile grids are encoded and decoded by a Variational Autoencoder. The architecture is purpose-built for 1D distribution data:

Figure 3: The Distribution VAE architecture. 1D CNNs encode quantile grids into a 32-dimensional latent space; the decoder reconstructs with guaranteed monotonicity.

Encoder

Three 1D convolutional layers with stride-2 downsampling and GELU activations, followed by adaptive average pooling. Projects to μ and log σ² via linear layers.

Conv1d GELU AdaptiveAvgPool

Decoder

Linear projection to a spatial feature map, then three transposed convolutions to upsample. Output is interpolated to the target grid size and made monotonic via start + cumsum(softplus(Δ)).

ConvTranspose1d Monotonic

4. Loss Function

The total training loss has two components, following the standard VAE framework:

Ltotal = Lrecon + β · DKL(q(z|x) || p(z))

Reconstruction Loss: Cramer Distance

The default reconstruction loss is the Cramer distance — the mean squared difference between input and reconstructed quantile functions. This is equivalent to the squared Wasserstein-2 distance when computed on quantile grids:

LCramer = (1/K) ∑i (Qinput(pi) − Qrecon(pi))²

Two additional distributional losses are available but disabled by default:

LossFormulaDefault WeightProperties
Cramer MSE on quantile grids 1.0 Smooth, differentiable, penalizes large deviations
Wasserstein-1 MAE on quantile grids 0.0 More robust to outliers
KL (quantile) log-ratio of quantile spacings 0.0 Sensitive to density differences

KL Divergence with β-Warmup

The latent-space KL divergence regularizes the encoder to produce a smooth, structured latent space. A β-warmup schedule linearly increases the KL weight from 0 to the target β=0.01 over the first 10 epochs, preventing posterior collapse early in training.

5. Training Pipeline

1
Data Preparation — Load AnnData (or generate synthetic Gaussian mixtures). Group cells by perturbation, extract per-gene expression distributions, convert each to a 256-point quantile grid. Split 90/10 train/val.
2
Training Loop — AdamW optimizer (lr=1e-3, weight_decay=1e-4) with cosine annealing LR schedule. Gradient clipping at norm=1.0. KL β-warmup over the first 10 epochs.
3
Checkpointing — Best model saved by validation reconstruction loss. Reconstruction snapshot plots every 25 epochs. Full training curves saved at end.
4
Encoding — After training, run the encoder on the full dataset to produce a fixed-size latent matrix: one 32-d vector per (gene, perturbation) pair. Output as AnnData .h5ad.
5
Evaluation — Compute Cramer, Wasserstein-1, and KS distances. Generate reconstruction plots, latent space PCA/UMAP, interpolation plots, and latent statistics.

10. Why This Matters

The Distribution Problem in Genomics

Traditional Perturb-seq analysis reduces each (gene, perturbation) pair to a single number (e.g., mean expression or log-fold-change). This discards critical information:

  • Bimodality — a perturbation might affect only a subpopulation of cells
  • Variance changes — a perturbation might increase expression noise without changing the mean
  • Skewness & tail behavior — subtle distributional shifts that mean/variance miss

What Distribution VAE Provides

By encoding the entire distribution into a 32-dimensional latent vector, this library preserves all distributional information in a format that standard ML models can consume directly. The learned representations can be used for:

  • Perturbation classification — which perturbation produced this expression pattern?
  • Gene program discovery — cluster distributions to find co-regulated genes
  • Dose-response modeling — interpolate in latent space between control and treated
  • Anomaly detection — identify unusual distributional responses
